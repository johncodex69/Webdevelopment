2. Lists

Lists are an important part of programming since they are so oftenly used.It is an advancement on first lessons
since instead of coding one set of information(a sring),one can code a huge set of information that works
together to give certain set of information to a user.

We learn to manipulate data in a list to make it fit different criterias.This includes asking for user input
then giving the user feedback depending on their input. The data can be programmed to give different
data according different scenarios where there are different inputs from the user(s).

There are a number of things that one can do to program a list to give vaying feedback on different inputs.These are;

  • Finding index of the list(Indexing)
  • Adding elements to the end of a list(Appending)
  • Arranging the list in ascending order(Sorting)
Just to name a few.
